# 🎓 EduDraft Studio

EduDraft Studio is a structured AI-powered platform for generating, formatting, and exporting high-quality educational materials.

It is designed for real classroom use - combining document intelligence, template systems, and diagram rendering into a single professional workflow.

---

## 🎯 What EduDraft Studio Does

EduDraft Studio transforms simple instructions into fully structured, classroom-ready outputs.

It enables educators to:

- Generate tests, exams, and worksheets  
- Create lesson notes and structured explanations  
- Build rubrics, marking guides, and memos  
- Apply institutional formatting automatically  
- Generate diagrams and visual aids  
- Export professional documents (DOCX and PPTX)  

---

## 🧠 Core System Architecture

### Template Engine
- Extracts structure and style from donor documents  
- Applies institutional formatting automatically  

### Diagram Engine
- Deterministic rendering where possible  
- AI fallback for complex diagrams (credit-controlled)  

### Draft Generation Pipeline
- Converts user input into structured drafts  
- Supports voice and text workflows
